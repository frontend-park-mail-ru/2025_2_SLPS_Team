    options Object Параметры пункта меню.
    Name Type Attributes Default Description
    label string Текст метки пункта меню.
    view string Идентификатор или название вида, связанного с пунктом.
    icon string | null <optional>
    		 null Название или путь иконки для пункта меню.
    isActive boolean <optional>
    		 false Флаг, указывающий, активен ли пункт меню.
    onClick function <optional>
    		 Функция, вызываемая при клике на пункт меню; получает `view` как аргумент.

    Type:
    Promise.<HTMLElement>
    Promise, который разрешается в HTML-элемент пункта меню.
    Example

  label: 'Главная',
  view: 'home',
  icon: 'home-icon',
  isActive: true,
  onClick: (view) => console.log('Перешли на', view)
}).then(menuItem => document.body.appendChild(menuItem));
